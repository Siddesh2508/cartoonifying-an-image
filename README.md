# Cartoonify Image Using Python

A computer vision project that converts normal images into cartoon-style images using OpenCV and Python.

## Features

- Upload image from system
- Webcam image capture
- Cartoonify effect
- GUI using Tkinter
- Save cartoon image

## Technologies Used

- Python
- OpenCV
- NumPy
- Tkinter
- Matplotlib
